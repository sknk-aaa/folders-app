ご質問

01 ブックマークのデータはどこに保存されますか？ +

すべてのブックマークデータはお使いのデバイス内にのみ保存されます。外部サーバーへの送信や、クラウドへのアップロードは一切行いません。

02 アプリを削除するとデータはどうなりますか？ +

アプリをアンインストールすると、保存されたすべてのブックマークデータが削除されます。大切なブックマークは事前にご確認ください。

03 Bookrest Pro を購入すると何ができますか？ +

ブックマークを100件以上保存できるようになります。またフォルダにPINロックを設定して保護できます。

一度購入すれば追加費用なしでずっとお使いいただける、買い切りプランです。

04 別のデバイスで Pro の購入を復元できますか？ +

はい。メニューの「プレミアム版を購入」から「購入を復元」をタップすると、同じ Apple ID で購入した Pro を復元できます。

05 URLスキームで外部からブックマークを追加できますか？ +

foldersapp://add?url=https://example.com の形式で呼び出すと、ブックマーク追加画面が開きます。Safari の共有シートからも追加できます。

06 不具合や要望はどこに送ればいいですか？ +

625.somq2525@gmail.com までご連絡ください。いただいたフィードバックはアプリの改善に活かします。
